Những bước để push code lên GitHub
Bước 1:
Tạo ra dự án mới trên github đây là phần remote repo

Bước 2:
Tạo dự án trên máy tính dùng để code và lưu trữ cục bộ

Bước 3:
Khởi tạo git: để khởi tạo một kho lưu trữ git cục bộ - Local repo
Bước 3.1:
Mở terminal

Bước 3.2:
config mặc định chọn git bash khi mở terminal

Bước 3.3:
git status : để kiểm tra trạng thái git

Bước 3.4:
git init : khởi tạo một kho lưu trữ Local

Bước 4:
add thư mục dự án vào staging area chờ commit
git add . : add tất cả thư mục
git add tên_thư_mục : add lẻ thư mục, tệp

Bước 5:
Commit và đẩy code lên GitHub
Bước 5.1
git commit -m "first commit" : sau khi commit code đẩy vào local repo chờ push lên GitHub
- first commit : là tên trong lịch sử

Bước 5.2
git branch -M main : do trên GitHub là main mà trong máy tính cục bộ là master nên cần cập nhật lại

Bước 5.3
git remote add origin Đường_dẫn_đến_dự_án : Remote đến máy chủ git

Bước 5.4
git push -u origin main : push code lên máy chủ Git



PUSH CODE LÊN DỰ ÁN CÓ SẴN

git remote add origin Đường_dẫn_đến_dự_án
git branch -M main
git push -u origin main

3. Clone repo về máy
Tiếp theo, việc của bạn bây giờ là hãy clone repo mới này về máy của mình bằng lệnh git clone địa_chỉ.

Ví dụ ở trên, chúng ta sẽ clone repo về máy bằng lệnh:

$ git clone https://github.com/ltnquang/ute.git

4. Push code lên git
Thêm/sửa/xóa các file/thư mục trên repo vừa clone về, sau đó lần lượt chạy từng lệnh sau:

$ git add

$ git commit -m "điền nội dung commit vào đây"

$ git push origin main




Git Branches Explained
Tạo, chuyển đổi và quản lí các nhánh dự án

git branch : kiểm tra cục bộ nhánh git trên máy tính
git branch -a : kiểm tra nhánh trên github và máy tính dấu * ở phía nhánh nào tức là đang đứng ở nhánh đó
git branch -c tên_nhánh_mới : tạo ra 1 nhánh mới nhưng vẫn ở nhánh hiện tại, chỉ tạo ở máy cục bộ
git checkout tên_nhánh : qua nhánh tên_nhánh
git checkout -b tên_nhánh_mới : tạo ra nhánh mới và qua nhánh đó luôn, chỉ tạo ở máy cục bộ
git push -u origin tên_nhánh : push tên_nhánh lên github
git branch -d tên_nhánh : không đứng ở tên_nhánh, xóa tên_nhánh khỏi máy cục bộ
Nếu tên_nhánh đang thực hiện commit nhưng vẫn muốn xóa tên_nhánh ta thực hiện: git branch -D tên_nhánh
git push -d origin tên_nhánh : xóa tên_nhánh khỏi github
